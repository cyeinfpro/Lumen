# 计费后台启用 Runbook

## 从 0 启用

1. 进入 Admin → 计费 → 概览。
2. 确认 USD→RMB 汇率,点击“初始化计费”。系统会自动生成强随机兑换码 secret。
3. 进入“定价”,确认 `billing.enabled` 已开启。
4. 检查尺寸定价表至少包含 `1k` / `2k` / `4k`,且每个档位都有价格。
5. 导入或编辑对话模型价格。
6. 回到“概览”,确认健康检查全部通过。
7. 进入“兑换码”,创建一张小额测试码。
8. 用普通 wallet 账号兑换,确认余额增加。
9. 提交一次测试生图或对话,在“用户钱包”里确认出现 hold / settle / charge 流水。

## 日常操作

- 改尺寸价格: Admin → 计费 → 定价 → 尺寸定价。阈值和价格会原子保存。
- 发码: Admin → 计费 → 兑换码。创建后立即复制明文或下载 CSV/TXT。
- 调账: Admin → 计费 → 用户钱包。搜索用户,打开详情后调账。
- 查异常: Admin → 计费 → 概览 → 最近审计。

## Secret 轮换

在 Admin → 计费 → 定价里点击“轮换”会由系统重新生成强随机 `billing.redemption_code_secret`。轮换会撤销所有未兑换码。轮换前先在兑换码列表确认是否还有活跃批次,并提前通知客服。
